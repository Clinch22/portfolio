# SmartShop

SmartShop is a premium static e-commerce storefront built for portfolio presentation with HTML5, CSS3, and vanilla JavaScript. It demonstrates a realistic shopping experience without a backend or framework.

## Features

- Responsive home, shop, product detail, cart, checkout, success, and 404 pages
- Product filtering, search, sorting, wishlist, and persistent cart with `localStorage`
- Cart drawer, quantity controls, order summaries, and demo checkout validation
- Semantic HTML, ARIA labels, keyboard-friendly modals, focus states, and reduced-motion support
- SEO basics: titles, meta descriptions, Open Graph content, favicon, and structured headings

## Structure

```text
SmartShop/
  index.html
  shop.html
  product.html
  cart.html
  checkout.html
  success.html
  404.html
  css/
  js/
  assets/
```

## Run Locally

Open `index.html` in a browser. No build step is required.

Some fonts, icons, and product photos load from Google Fonts, Font Awesome, and Unsplash, so an internet connection gives the best visual result.
